# Calculator
A modern, mobile-inspired scientific calculator built using HTML, CSS, and JavaScript, inspired by the Realme calculator UI.
It combines advanced scientific calculations with a multi-category unit converter, smooth animations, and an intuitive user experience — all running fully in the browser.

This project focuses on UI/UX polish, clean logic, and real-world utility, without using any external frameworks.

The calculator supports:

• Basic arithmetic

• Scientific functions

• Angle modes (Radians/Degrees)

• Mathematical constants

• Factorials and powers

In addition, it includes a slide-in unit converter drawer that allows users to convert values across multiple domains such as length, weight, area, volume, temperature, speed, pressure, power, number systems, and currency.

### Key Features
• Basic operations: + − × ÷ %

• Scientific functions:

        • sin, cos, tan
    
        • log, ln
    
        • √, factorial !, inverse
    
        • Power operator ^
    
        • Constants: π, e

• Bracket support ( )

• Radians / Degrees toggle

• Animated function preview before calculation

• Smart input handling & error prevention
